# 前端
## HTML
### 标签

```
<a href> 超文本链接
<p> 段落
<li>定义无序列表
<img src> 图片src表示超链接
<dl>定义一个描述列表
<dt>定义一个项目
<dd>项目描述
<link rel="stylesheet" href="home.css"> 定义一个外联css
<a href="maito:xxxx@163.com"></a>
<br>换行
```    


### 属性 

```
#标识/name
id
#分块的名字
class
#事件会在元素被点击时发生，主要是执行js。
onclick
````

## CSS

```
color
text-align 文本的对齐方式
text-decoration  文本的装饰，主要是用来删除链接的下划线
text-indent   文本缩进
word-spacing   字间距

font-family  字体，可以在后多写几个，如果浏览器不支持，他会自动使用下一个
font-size  字体的大小

a:link {color:#000000;}      /* 未访问链接*/
a:visited {color:#00FF00;}  /* 已访问链接 */
a:hover {color:#FF00FF;}  /* 鼠标移动到链接上 */
a:active {color:#0000FF;}  /* 鼠标点击时 */

div {
    width: 300px;
    border: 25px solid green;
    padding: 25px;
    margin: 25px;
}
盒子的总宽度为300
```
